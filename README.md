# Đồ án: Xây dựng Website Sàn Tuyển Dụng
Sinh viên thực hiện: Nguyễn Thành Khoa Đăng  
MSSV: 24410136  

## Mục tiêu
Xây dựng một website sàn tuyển dụng cho phép:
- Nhà tuyển dụng đăng tin tuyển dụng
- Ứng viên tìm việc, xem chi tiết job, và ứng tuyển
- Dashboard thống kê đơn giản (biểu đồ)
- Một số chức năng nâng cao như gợi ý kỹ năng (simple NLP)

## Công nghệ dự kiến sử dụng
- Python 3.10+
- Django 4.x
- SQLite
- Bootstrap 5 (frontend)
- Chart.js
